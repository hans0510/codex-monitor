<threat_model>
Assets: local usage output and local Codex filesystem paths.
Threats:
- CLI accidentally falls back to account APIs or network calls.
- CLI scans broad directories when default Codex home is missing.
- Error output leaks unrelated file contents.
Mitigations:
- CLI calls only the core local discovery and aggregation functions.
- `--codex-home` is a path override, not a recursive arbitrary data import beyond the core allowlist.
- Error messages report missing paths and diagnostics, not file contents or credentials.
Residual risk: Users may point `--codex-home` at an unexpected directory; allowlisted child scanning limits blast radius.
</threat_model>
